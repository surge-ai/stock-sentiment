## Columns

* `Stock Ticker`: The stock ticker identified in the Tweet, including the `$`
* `Tweet Text`: The text of the tweet
* `Sentiment`: The sentiment, either `Positive` or `Negative`
* `Tweet URL`: The URL to the original tweet.
